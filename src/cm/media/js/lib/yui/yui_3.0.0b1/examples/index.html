Yahoo! User Interface Library (YUI): Index of Library Examples

Note: This is YUI 3.x. Looking for YUI 2.x?

Official YUI Examples

The YUI Library 3.x ships with a series of 128 examples that illustrate the implementation of its components. The examples can be starting points for your exploration, code snippets to jump-start your own programming, or simply inspiration as to how various interaction patterns can be enabled in the web browser via YUI.

This page allow you to explore these examples component-by-component; on this page you'll find the full index of library examples with the link and short description of each.
